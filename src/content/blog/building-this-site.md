个人博客最重要的不是一开始就很复杂，而是足够稳定、足够容易更新。

GitHub Pages 很适合作为第一版过渡方案：

- 免费托管静态网站
- 可以绑定自己的域名
- 和 Git 仓库天然结合
- 适合长期维护 Markdown 文章

后续如果需要更强的预览、评论、统计或边缘部署能力，也可以迁移到 Vercel 或 Cloudflare Pages。
